# Matilda BETA Sensitivity

This repository archives code and documentation for an undergraduate research project testing how uncertainty in the terrestrial CO2 fertilization parameter, BETA, affects Hector/Matilda model outputs.

The project focuses on a sensitivity analysis that varies BETA across a full prior distribution, running Hector simulations through Matilda, and comparing resulting changes in atmospheric CO2, temperature, and carbon cycle behavior.

## Research Question

How sensitive are Hector/Matilda climate projections to uncertainty in the terrestrial CO2 fertilization parameter BETA?

## Background

The BETA parameter represents the strength of terrestrial CO2 fertilization in Hector. A higher BETA value generally informs a stronger plant productivity responses to rising CO2, which can increase land carbon uptake, leading to a negative warming feedback. A lower BETA value represents weaker CO2 fertilization and may result in more CO2 remaining in the atmosphere, a primary source of amplified warming trends.

Because atmospheric CO2 influences global temperature, uncertainty in BETA may affect future warming projections. This project aims to use Hector and Matilda to test the size and importance of that effect.
